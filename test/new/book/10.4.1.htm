10.4.1                移动掠扫算法

移动掠扫算法flash演示

     移动掠扫算法使用一个基本平面FACE，并指定一个移动方向（dx,dy,dz）。其中dx,dy,dz分别表示移动距离的三上分量，通过欧拉操作，自动生成一个多面体的半边数据结构。算法如程序10.1所示。该程序执行过程中，数据结构的逐步生成及各变量的位置变化如图10.4所示。其中（a）~(f)图与程序中标记位置情况对应。

 

/ *  procedure sweep is a translation sweeping routine

It takes a face fac of a solid, and sweeps it along

a vector [dx, dy, dz ].

* / 

void sweep(fac, dx, dy, dz)

Face * fac;

Float dx, dy, dz;

{

    Loop     * l;

HalfEdge*first,  *scan;

Vertex    * v;

 

lgetmaxnames(fac->fsolid);

l = fac->floops;

while(l)

{

first = l->ledg; / * a * /

scan = first->nxt;

v =scan->vtx;

lmev(scan, scan, ++maxv,

v->vcoord[0]+dx,

v->vcoord[1]+dy,

v->vcoord[2]+dz);    / * b * /

 

while(scam ! = first)

{

    v = scan->nxt->vtx;

lmev(scan->nxt, scan->nxt, ++maxv,

v->vcoord[0]+dx,

v->vcoord[1]+dy,

v->vcoord[2]+dz);    / * c * /

lmef(scan->prv, scan->nxt->nxt, ++maxf); / * d * /

scan = mate(scan->nxt)->nxt;

}     / * e * /

lmef(scan->prv, scan->nxt, ++maxf);

l = l->nextl;

}

}

/ *  end of sweep * /
