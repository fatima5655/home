3.1.1 二维图形的几何变换

一、基本变换

⒈ 平移(Translation)

平移是将对象从一个位置(x, y)移到另一个位置(x¢, y¢)的变换(图3.1)。Tx=x¢-x，Ty=y¢-y称为平移距离。平移变换的公式为

                                                             图3.1 平移
x¢ = x+Ｔx，y¢ = y+Ｔy

 

⒉ 旋转(Rotation)

旋转是以某个参考点为圆心，将对象上的各点(x, y)围绕圆心转动一个逆时针角度q，变为新的坐标(x¢, y¢)的变换(图3.2)。当参考点为(0, 0)时，旋转的公式为

x¢ = rcos(a+q) = rcos acos q-rsin asin q

y¢ = rsin(a+q) = rsin acos q+rcos asin q

因为x=rcos a，y=rsin a，所以上式可化为

x¢ = xcos q-ysin q

y¢ = ycos q+xsin q

如果参考点不是(0, 0)，而是任意一点(xr, yr)，那么，绕(xr, yr)点的旋转由3个步骤完成：

1)    将对象平移Tx=-xr，Ty=-yr；

2)    按式(3.2)作旋转变换；

3)    平移Tx=xr，Ty=yr。组合这3个步骤的计算公式为

x¢ = xr+(x-xr)cos q-(y-yr)sin q

 图3.2  旋转
y¢ = yr+(y-yr)cos q+(x-xr)sin q

 

⒊ 变比(Scaling)

变比是使对象按比例因子(Sx, Sy)放大或缩小的变换(图3.3)。变比计算公式为

 图3.3  变比
x¢ = x ・ Sx，y¢ = y ・ Sy

从图3.3可见，按式(3.3)作变比变换时，不仅对象的大小变化，而且，对象离原点的距离也发生了变化。如果只希望变换对象的大小，而不希望变比对象离原点的距离，则可采用固定点变比(scaling relative to a fixed point)。以a为固定点进行变比的方法是：

1）    作平移Tx=-xa，Ty=-ya；

2）    按式(3.3)作变比；

3）    作1)的逆变换，即作平移Tx=xa，Ty=ya。

                     图3.4  反射
当比例因子Sx或Sy小于0时，对象不仅变化大小，而且分别按x轴或y轴被反射。

图3.4(a)表示当Sy=-1，Sx=1时的变化，此时按x轴反射。图3.4(b)表示当Sy=1，Sx=-1时的变比，此时按y轴反射。图3.4(c)示出当Sx=-1，Sy=-1时按原点(0, 0)反射的情况。

二、变换矩阵

上述3种基本变换公式都可以表示为3×3的变换矩阵和齐次坐标相乘的形式。

⒈ 平移的矩阵运算表示为

                                     (3.1)

简记为p¢=p・T(Tx, Ty)。其中，p¢=［x¢  y¢  1］，p=［x  y  1］。

表示平移矩阵。

⒉ 旋转的矩阵运算表示为

                                  (3.2)

简记为p¢=p×R(q)，其中R(q)表示旋转矩阵。

⒊ 变比的矩阵运算表示为

                                       (3.3)

简记为p¢=p×S(Sx, Sy)，其中(Sx, Sy)表示变化矩阵。

三、级联变换(Composite Transformation)

一个比较复杂的变换需要连续进行若干个基本变换才能完成。例如，围绕任意点(xr, yr)的旋转，就要通过3个基本变换T(-xr, -yr)，R(q)，T(xr, yr)才能完成。这些由基本变换构成的连续变换序列称为级联变换。

变换的矩阵形式使得级联变换的计算工作量大为减少。以绕任意点旋转变换为例，本应进行如下3次变换：

                          p¢ = p・T(-xr, -yr)                        (3.4)

                            p² = p¢・R(q)                          (3.5)

                          p²¢ = p²・T(xr, yr)                         (3.6)

将式(3.4)、(3.5)代入式(3.6)，得

p²¢ = p・T(-xr, -yr)・R(q)・T(xr, yr)

令Tc=Ｔ(-xr, -yr)・R(q)・T(xr, yr)则有

p²¢ = p・Tc

Tc称为级联变换矩阵。由上面推导可知在计算级联变换时，首先可将各基本变换矩阵按顺序相乘，形成总的级联变换矩阵Tc。然后，坐标只需与Tc相乘一次，便可同时完成一连串基本变换。所以，采用级联变换矩阵，大大节省了坐标乘法所耗费的运算时间。

四、二维几何变换的指令

二维基本变换的指令有3条，分别执行建立变换矩阵、积累变换和坐标变换等3种操作。

⒈ 建立变换矩阵的指令为

creat_transformation_matrix(xf, yf, Sx, Sy, xr, yr, a, Tx, Ty, matrix)；

其中

xf、yf是固定点变比的固定点坐标；

Sx、Sy是固定点变比的变比因子；

xr、yr是任意点旋转的任意点坐标；

a是旋转的角度；

Tx、Ty是平移的距离；

matrix是所建立的变换矩阵。

此指令按参数的先后次序依次进行变换，即执行如下操作

matrix=Ｔ(-xf, -yf)×S(Sx, Sy)×T(xf, yf)×T(-xr, -yr)×R(a)×T(xr, yr)×T(Tx, Ty)

⒉ 积累变换的指令为

accumulate_transformation_matrix(matrix1, matrix2, matrix);

其中，matrix1、matrix2是两个输入矩阵，matrix是返回矩阵。它们都是3×3矩阵。该指令完成如下功能

matrix = matrix1×matrix2

⒊ 坐标变换的指令为

set_segment_transformation(Id, matrix);

其中Id是对象的编码(segment的概念在第五章介绍)，matrix是几何变换矩阵。该指令完成如下功能

［x¢id  y¢id  1］=［xid  yid  1］×matrix

其中xid、yid是Id对象的所有坐标值。
