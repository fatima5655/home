3.3.1  二维图元输出

用户使用图形软件时，首先要让计算机知道需要画什么。图元是图形软件用于组织和操作画面的最基本素材。一幅画面由图元组成，图元是一组最简单的、最通用的几何图形或字符。

图形软件常用的图元有line(直线段)、polyline(折线)、text(字符串)、fill area(色块)、polymarker(符号的轨迹)、move(现行位置的定位)、cell array(像素矩阵)、circle(圆)。每种图元都有一系列系数描述其性质。在图形软件包中这些参数由两类命令定义。一类是输出图元命令，它描述了图元的种类及其最重要、也是变化最频繁的参数，如直线的端点坐标。另一类是图元性质定义命令，它描述了图元的其余性质，如直线的宽度、线型等。从实现的角度来看，用户定义的二维图元的窗口区到视图区的输出过程如下所示：

输出图元的命令介绍如下：

⑴ move(x, y)

定义当前位置(Current Position)为(x, y)。启动图形软件后，当前位置自动定义为(0, 0)。用户需用此命令将当前位置移到作图的起始点。

⑵ line(x, y)

以当前位置为起点，(x, y)为终点，画一条直线段。当(x, y)等于当前位置时，此命令实现了画点功能。

⑶ polyline(n, x_array, y_array)

参数中的x_array和y_array是两个一维数组，每个数组的长度为n。本命令画一条有n+1个顶点(包括两个端点)的折线。第一个顶点在当前位置，其余n个顶点的位置由x_array和y_array中的值(分别代表顶点的x、y坐标)决定。

例如，设有两个数组内容如下：

xlist＝｛3, 5, 5, 2, 2, 5｝，ylist＝｛2, 2, 5, 5, 7, 7｝

下面两条命令画出图3.16中的“5”字。

move(2, 3);

 图3.16  输出图元
polyline(6, xlist, ylist);
